# Smart Q - Beauty & Wellness Queue (KBZPay Mini App)

Smart Q is a high-fidelity prototype of a Beauty & Wellness booking and queue management platform, designed to run as a KBZPay Mini App.

## Features

- **Mobile-First Experience**: Renders inside a realistic smartphone frame.
- **Smart Booking**: Multi-step flow to select services and staff.
- **Live Queue**: Real-time token tracking with estimated wait times.
- **Wait-time Commerce**: Contextual product and offer recommendations while waiting.
- **Loyalty Integration**: Earn points and track tier progress.
- **Merchant Dashboard**: Full control over queue, staff, and branch analytics.

## Tech Stack

- **Frontend**: React, Vite, TypeScript, Tailwind CSS, Framer Motion.
- **Icons**: Lucide React.
- **Routing**: React Router DOM.
- **API**: Mock service layer (ready for AppCube integration).

## Setup & Deployment

1. **Install Dependencies**: `npm install`
2. **Run Development**: `npm run dev`
3. **Build for Production**: `npm run build`
4. **Deploy to Netlify**:
   - Build Command: `npm run build`
   - Publish Directory: `dist`
   - Add a `_redirects` file in `public/` with: `/* /index.html 200`

## Demo Script (3 Minutes)

1. **Introduction (30s)**:
   - "Welcome to Smart Q, the future of Beauty & Wellness bookings in the KBZPay ecosystem."
   - Show the app running in the phone frame.
   - "Our goal is to eliminate wait-time frustration and boost merchant revenue."

2. **Customer Journey (90s)**:
   - **Discovery**: Browse branches like 'Glow Up Studio'.
   - **Booking**: Select 'Hair Cut', pick 'Su Su' as the stylist.
   - **Payment**: Pay the 200 Ks booking fee (MDR) to secure a slot.
   - **Queue**: Show the live token (e.g., A12). Explain 'Smart Arrival' notifications.
   - **Wait-time Commerce**: "While waiting, I see a 10% discount on Argan Oil. I can claim it right here."

3. **Merchant View (45s)**:
   - Switch to the Merchant Dashboard using the toggle.
   - "Merchants see their live queue, call the next customer, and monitor branch analytics in real-time."
   - Show the 'Analytics' tab with revenue and efficiency stats.

4. **Conclusion (15s)**:
   - "Smart Q turns idle wait time into commerce opportunities while providing a seamless experience for both customers and merchants."
